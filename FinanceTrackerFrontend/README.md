# 🏦 Modern Finance Tracker (AI-Powered)

Experience a smarter way to manage your wealth. This is a premium, feature-rich Finance Tracking application built with React, Vite, and Tailwind CSS. It features AI-driven insights, gamified health scoring, and voice-activated data entry.

## 🌟 Key Features

*   **🤖 AI Financial Advisor**: A built-in chat assistant that provides real-time spending analysis, budget predictions, and personalized savings tips.
*   **🌡️ Financial Health Score**: A gamified 0-850 scoring system (similar to credit scores) that analyzes your Savings Rate, Budget Adherence, and Consistency.
*   **🔥 Spending Heatmap**: A GitHub-style activity grid to visualize your daily spending patterns over time at a glance.
*   **🎙️ Voice-Activated Entry**: Hands-free transaction logging using the Web Speech API. Just say "Spent 500 rupees on dinner" and let the AI do the rest.
*   **📊 Dynamic Dashboards**: Interactive charts (Area & Pie) for monthly expense trends and category breakdowns.
*   **📄 PDF Reports**: Generates branded, professional financial reports including charts and a transaction history in one click.
*   **🌓 Smart Dark Mode**: Fully responsive UI with automated dark/light mode detection for a premium eye-care experience.

## 🛠️ Tech Stack

*   **Frontend**: React.js, Vite
*   **Styling**: Tailwind CSS v4, Lucide React (Icons)
*   **Charts**: Recharts
*   **Reporting**: jsPDF
*   **API**: Axios (Integrated with Spring Boot Backend)
*   **Animation**: Tailwind Animate

## 🚀 Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/FinanceTrackerFrontend.git
    cd FinanceTrackerFrontend
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Connect to Backend**: Ensure your [FinanceTrackerApi](https://github.com/tapan2004/FinanceTrackerApi) is running locally on port 8080 (or your configured port).
