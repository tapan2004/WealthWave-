# WealthWave

This repository contains both the backend API and the frontend UI for the WealthWave Finance Tracker application.

## Repository Structure

- **[WealthWaveApi](./WealthWaveApi/)**: Node.js & Express backend API.
- **[WealthWaveUI](./WealthWaveUI/)**: React & Vite frontend UI.

## Getting Started

### Backend
1. Navigate to `WealthWaveApi` directory:
   ```bash
   cd WealthWaveApi
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file and set the environment variables.
4. Run the development server:
   ```bash
   npm run dev
   ```

### Frontend
1. Navigate to `WealthWaveUI` directory:
   ```bash
   cd WealthWaveUI
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
